■ 今日判断
今天最重要的事是区分两类信号：「噪声」和「新事实」。非农数据触发的恐慌是噪声——美国餐饮业世界杯季节性就业不代表经济过热，沃什大概率不会在首次议息会议就加息。真正的新事实有三个：一是SpaceX-Google算力月租9.2亿美元落地，算力租赁从概念变成现金流；二是SoCAMM降容被误读为利空，实质是供给告急的反面信号，存储涨价路径更加确定；三是中天科技中标15亿MPO光纤跳线，国内数据中心光纤需求翻倍趋势有了具体合同数字支撑。操盘层面：周一低开后，AI产业链的核心逻辑无任何改变，错杀是加仓机会；被动元件涨价继续扩散（电感7月再涨），是与美股情绪无关的国内独立信号，可重点关注。

▲ 逻辑反转 存储 / DRAM 错杀即机会
SoCAMM降容是供不应求的信号：英伟达减容换出货量，存储总需求反而上调10-20%
韩国Meritz证券 · 中泰电子 · 0607
SemiAnalysis报道Vera Rubin NVL72 SoCAMM从192GB降至96GB，美光大跌，SK/三星跌5-8%。但多方产业链调研确认：降容原因是MU供给不足——在内存极度紧张情况下，英伟达选择减少单机架容量以放量更多台整机。96GB模块订单量提升近6倍，64GB模块+50%，SoCAMM整体出货总量预计上调10-20%。基板规格、核心原料、后端制程均无重大变更，供应链全链条受益出货量提升。

唯一MU为SoCAMM独家量产

为什么这次下跌是误读？任何财大气粗的CSP（谷歌/亚马逊/微软）不会为省80万美元而选择少装27TB内存。英伟达降容的唯一合理解释是：拿不到足够的MU产能——装满55TB只能出100台，装28TB就能出200台。在供不应求的环境下，减容是放量的手段，不是需求下降的信号。

存储周期位置在哪里？韩国Meritz明确：DRAM上行周期目前处于"大腿"区间，远未到"肩部"减持时机。NAND Q2合约价+70-75%，DRAM Q2+58-63%，Q3伯恩斯坦上修至+60%环比。长鑫IPO提交注册临近，有望持续带动国内存储板块情绪。兆易创新作为长鑫核心采购方，26H1 DRAM采购额已超过25年全年。

产业链受益路径。降容后96GB规格成主力，出货量大幅上升，整条产业链—基板/原料/后端制程—受益量增。下一代产品趋势：增加基板层数+升级堆叠架构+大范围低损耗材料，推动高附加值基板与高端原材料迭代。企业级SSD同样受益（减少DRAM省出预算转向NAND）。

兆易创新 长鑫存储 中电港 江波龙 北京君正 澜起科技 深科技 长电科技
→ 所以 周一存储板块跟随美股下跌是错杀机会，核心逻辑（供不应求+Q3继续涨价）没有任何改变，只是被市场误读强化了。兆易创新（长鑫IPO+Flash涨价+DRAM采购翻倍三重催化）是今日存储板块预期差最大的国内标的，知识库显示该逻辑已连续跟踪，本期有SoCAMM事件提供新的验证节点。

SoCAMM（Small Outline Compression Attached Memory Module）
英伟达为AI服务器设计的新型内存模块规格，采用LPDDR5X存储颗粒，体积小于标准DIMM，通过压缩附着方式连接主板，实现更高带宽密度。在Vera Rubin NVL72中承担CPU侧缓存和KV cache存储职责。目前由美光（Micron）独家量产供应，是整个存储产业链中供给最紧张的环节。
为何今日重要：SoCAMM从192GB降至96GB的新闻引发市场误读导致存储板块大跌，但实质是供给不足→降容放量→总出货量反升10-20%的正面信号，是今日最值得关注的预期差所在。
MPO光纤跳线（Multi-fiber Push On）
一种高密度多芯光纤连接器，单个接口可同时连接12、24或更多芯光纤，比传统LC/SC连接器大幅提升数据中心布线密度和速度。在AIDC Scale-out架构中用于机架间高速互联，是今日AI数据中心光纤需求最直接的终端产品形态之一。
为何今日重要：中天科技中标15.18亿元MPO光纤跳线合同，是国内数据中心光纤需求翻倍趋势首次以具体合同金额落地，验证了光纤板块的量价齐升逻辑。
外汇脉冲（FX Impulse）
国泰海通宏观构建的新流动性指标，用于捕捉跨境资金流入带来的"高能流动性"变化。传统信用指标（社融增速/信贷脉冲）在当前阶段与股市背离，原因是流动性主要通过外汇渠道而非信用渠道投放。中国出口竞争力提升→贸易顺差扩大→结汇资金回流→股债汇三牛，是外汇脉冲指标的核心传导路径。
为何今日重要：理解这个框架有助于解释为何A股在信用收缩期仍能走强，以及为何中美权益市场联动性上升——中国资产对美元和美债利率的敏感度同步上升，非农数据冲击的传导路径由此可解。
硬碳负极（Hard Carbon Anode）
钠离子电池最主流的负极材料，由椰壳、树脂或煤炭在高温下碳化制成，具有无序的乱层结构，能够储存钠离子。相比石墨负极（锂电池专用），硬碳对钠离子的嵌入/脱嵌更友好。核心性能指标：可逆容量（340mAh/g为先进水平）和首效（92%为优异水平）。资源端：椰壳硬碳仅能支撑50-60GWh需求，万华化学等已转向树脂基/煤基路线。
为何今日重要：宁德明确26年底钠电电芯成本追平铁锂，硬碳负极的降本路线（从椰壳向树脂基/煤基转型）是实现这一目标的核心技术路径，万华化学等国内企业的提前布局值得关注。
