→ 所以 今日建滔/电子布的调整是情绪误判，不是基本面变化。PTFE无布方案要到2027年Q2才批量供货，当下电子布的结构性缺口完全不受影响。6月提价正在落地，这是买入布局窗口而非减仓信号。

MLCC现货价涨2-3倍 vs 原厂未涨价
——分销商商络电子吃掉全部差价
产业链洞察 · 0601
今日发现MLCC关键逻辑漏洞：MLCC原厂现阶段还没有正式涨价，但现货价格已经涨了2-3倍——这中间的价差完全被分销商赚走。国内最大MLCC分销商商络电子（库存13亿+）处于最优位置：囤货期间已按更低价格锁定库存，现在按现货价出货，吃掉全部涨价差价。

申万电子量化：假设MLCC现货价涨价50%/100%/150%，国瓷材料（MLCC粉体，70%国内市占）可分别增厚净利润5.5/11/16.5亿元。年中二次涨价预期仍强，高容产能缺口将持续至2027年。光设备板块26Q1-Q2集体订单3-5倍增长，当前节点建议关注新技术（CPO/NPO）耦合（科瑞技术）和测试（联讯仪器）两个方向。
→ 所以 在原厂正式涨价落地之前，分销商（商络电子）是MLCC涨价周期中最快兑现利润的环节。原厂涨价一旦落地，差价收敛，利润重新回归三环/风华/昀冢。这是时序逻辑，不是谁胜出的问题——分销商先享受，原厂随后接棒。

RTX Spark（原N1X）
英伟达与微软、联发科联合定制的ARM架构笔记本SoC，今日GTC Taipei正式发布，以"RTX Spark"品牌命名。核心规格：6144 CUDA核心（Blackwell架构）+ 20核Grace CPU（10性能+10效率）+ 128GB LPDDR5X统一内存 + NVLink-C2C 600GB/s + 700亿晶体管（3nm工艺）。端侧AI算力180-200 TOPS，可在本地运行70B参数大模型。秋季发布，合作OEM涵盖微软、戴尔、联想、惠普、华硕、微星。这是英伟达首款面向笔记本PC的芯片，标志着"Win-ARM-CUDA"时代开启，PC估值逻辑从周期品升级为AI基础设施。
为何今日重要：RTX Spark让PC行业的估值逻辑发生根本性转变——从"每几年换代"的消费硬件，变为企业级AI基础设施节点。OEM（联想/戴尔）开辟对抗MacBook Pro的高端产品线；ODM（工业富联）毛利率和准入门槛提升；PCB（胜宏/鹏鼎）和散热（英力股份同步受益）单机价值量大幅提升。
Groq LPX Tray（Groq 3 Logic Processing Unit）
英伟达在GTC展示的Vera Rubin系统中全新增加的机柜单元，Groq 3 LPX专为大规模推理（Inference）工作负载设计，与GPU协同工作——GPU负责训练和通用计算，LPX负责低延迟、高吞吐的Token生成推理任务。技术指标：与Vera Rubin NVL72配合，35×高吞吐推理（相比上代），超低延迟设计，是Agent AI时代"快速响应"需求的专用硬件。这是英伟达"AI Factory"全栈体系中的推理专用层，也是对"推理侧算力"单独定价的重要信号。
为何今日重要：LPX Tray的出现说明英伟达已将推理（Inference）从训练（Training）中拆分出来，作为独立的系统级单元进行优化。这意味着AI数据中心将同时部署训练GPU + 推理LPX的混合架构，推理算力需求将产生独立的采购体量，是继GPU之后的第二个规模化硬件品类。
AI Factory（AI工厂）
Jensen Huang今日GTC keynote中提出并系统阐述的新范式：AI数据中心不再是"IT基础设施成本中心"，而是"工业生产单元"——输入Token（算力、能源、数据），输出AI产品（token、推理结果、智能服务），产生可量化的商业价值。这是AI从"研究阶段"到"商业化阶段"的认知切换，也是Jensen Huang说"AI now is a profit generator"的理论基础。AI Factory的核心是全栈协同设计（Extreme Co-design）：计算/存储/网络/电力/冷却的系统级优化，而非各自独立采购的最优。
为何今日重要：AI Factory叙事的确立，意味着AIDC从"卖给IT部门"升级为"卖给CFO"——投资回报率变得可量化，采购决策的评估逻辑随之改变。对国内算力链（浪潮信息/中科曙光）和电力设备（四方股份/阳光电源）来说，这是从"概念"转向"商业模型"的信号。
氟化液（Fluorinated Liquid / FC类冷却液）
以全氟聚醚（PFPE）、氢氟醚（HFE）、低聚体等为主要成分的电子级冷却液，用于浸没式液冷（服务器直接浸泡）和精密设备清洗（半导体制造中的精密仪器清洗）。特点：化学惰性、不导电、低表面张力、优异的热传导性、与电子器件兼容性好。2025年3M因环保问题退出市场（涉及PFPE+HFE产能1.2万吨），形成重大供给缺口。全氟聚醚当前价格50万元/吨，氢氟醚20万元/吨，国产替代机会显著，新宙邦、巨化股份、金石资源（通过诺亚布局低聚体）是国内主要玩家。
为何今日重要：浸没式液冷是数据中心PUE提升的最终方案，随着AI服务器功率密度持续攀升，氟化液从"可选冷却方案"变为"高密度AIDC必选"。3M退出留下的1.2万吨缺口，叠加液冷渗透率快速提升，将推动氟化液需求和价格同步上行。
